# Furniture3D_Visualization
A web-based interactive 2D/3D spatial visualization tool for furniture retailers, built with the MERN stack and Three.js.
## 📌 Project Overview
**Furniture 3D** is a web-based interactive software application developed for modern furniture retailers. Designed to be used by in-store designers during client consultations, it bridges the gap between imagination and reality. The application allows users to configure accurate room dimensions, plan layouts using an intuitive 2D drag-and-drop interface, and instantly generate a highly realistic 3D spatial visualization.

## ✨ Key Features
* **📐 Room Configuration:** Define exact width, length, shape, and floor base colors.
* **🗺️ 2D Floor Planning:** Interactive 2D workspace (`react-konva`) to drag, drop, rotate, and arrange furniture top-down architectural symbols.
* **🌐 Instant 3D Rendering:** One-click conversion to a 3D environment (`@react-three/fiber`) complete with realistic sunlight simulation, shadows, and PBR materials.
* **💾 Project Management:** Auto-saving functionality and a comprehensive Designer Dashboard to view, edit, or delete client projects.
* **🛡️ HCI & UX Optimized:** Built with clear system status feedback, error prevention (e.g., duplicate design names), and an accessible luxury dark-mode UI.

## 🛠️ Technology Stack
This project utilizes the **MERN** stack along with advanced web graphics libraries:
* **Frontend:** React.js, Tailwind CSS, React-Router-Dom
* **2D Graphics:** React-Konva
* **3D Rendering:** React-Three-Fiber, Drei, Three.js
* **Backend:** Node.js, Express.js
* **Database:** MongoDB, Mongoose
* **API Communication:** Axios
